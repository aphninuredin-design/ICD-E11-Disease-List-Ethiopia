# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest  | ✅ Yes             |
| Older   | ⚠️ Best effort     |

## Reporting a Vulnerability

**Please do not open a public issue for security vulnerabilities.**

Instead, email security concerns to: [Your contact email]

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if available)

We will:
1. Acknowledge receipt within 48 hours
2. Assess the severity
3. Provide updates on remediation
4. Credit you in the fix (if desired)

## Security Considerations

This PWA stores user data locally in the browser cache. Users should:
- Clear app cache if shared device
- Review browser storage settings
- Keep device updated with latest OS security patches
