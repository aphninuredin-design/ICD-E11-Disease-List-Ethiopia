# 🇪🇹 ICD-E11 Disease List Ethiopia (Primary Hospital Edition)

This project is a dedicated digital reference tool for the **Ethiopia Simplified Version of the International Classification of Diseases (ICD-11)**. It is designed to assist healthcare providers in Ethiopian primary hospitals to quickly search, identify, and apply standardized medical codes.

## 🎯 Project Goals
* **Accuracy:** Reduce coding errors by providing the official ESV_ICD11 list.
* **Efficiency:** Enable rapid search by disease name, category, or code.
* **Accessibility:** A mobile-first Progressive Web App (PWA) that works **offline** in clinical settings with limited internet.

## ✨ Key Features
* 🔍 **Smart Search:** Instant filtering across hundreds of diseases.
* 📁 **Categorized Data:** Organized by medical chapters (Infectious, Respiratory, etc.) as per the MoH manual.
* 📋 **One-Click Copy:** Quickly copy codes for use in Electronic Medical Record (EMR) systems.
* 📱 **PWA Ready:** Installable on Android and iOS devices.

## 🛠 Tech Stack
* **Frontend:** HTML5, Tailwind CSS (for modern UI).
* **Logic:** Vanilla JavaScript.
* **Data Source:** Ministry of Health - Ethiopia (Primary Hospital Edition PDF).
* **Deployment:** GitHub Pages.

## 🚀 How to Use
1. Open the app link on your smartphone browser.
2. Select "Add to Home Screen" to install it as an app.
3. Start searching for diseases even when offline.
