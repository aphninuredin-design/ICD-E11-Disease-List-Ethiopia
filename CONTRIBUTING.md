# Contributing to ICD-E11 Disease List Ethiopia

Thank you for your interest in contributing to this project! This tool serves important healthcare needs in Ethiopia, and we welcome contributions.

## How to Contribute

### Reporting Issues
- **Found a bug?** Open an issue with:
  - Clear description of the problem
  - Steps to reproduce
  - Expected vs actual behavior
  - Screenshots if applicable

- **Suggestion for improvement?** Open an issue with "Enhancement" label

### Making Changes

1. **Fork** the repository
2. **Create a branch** for your changes:
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. **Make your changes** and test thoroughly
4. **Commit** with clear messages:
   ```bash
   git commit -m "Add: description of your changes"
   ```
5. **Push** to your fork
6. **Create a Pull Request** with:
   - Clear title and description
   - Reference to related issues
   - List of changes made

### Guidelines

- Keep changes focused and atomic
- Test in multiple browsers and devices (especially mobile)
- Ensure PWA functionality is maintained
- Update documentation if needed
- Follow existing code style and conventions

## Data Corrections

If you notice errors in the ICD-11 data:
1. Verify against the official MoH Ethiopia documentation
2. Create an issue with evidence
3. Submit a PR with the correction

## Questions?

- Review existing issues and documentation first
- Open a discussion issue for general questions
